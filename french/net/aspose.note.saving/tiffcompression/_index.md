## TiffCompression enumeration

Spécifie le type de compression à utiliser lors de l'enregistrement d'un document au format TIFF.

```csharp
public enum TiffCompression
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| None | `1` | Spécifie aucune compression. |
| Rle | `2` | Spécifie la compression RLE. |
| Ccitt3 | `3` | Spécifie le codage de télécopie CCITT Groupe 3. |
| Ccitt4 | `4` | Spécifie le codage de télécopie CCITT Groupe 4. |
| Lzw | `5` | Spécifie la compression LZW. |
| PackBits | `32773` | Spécifie la compression Macintosh RLE. |
| Jpeg | `7` | Spécifie la compression de compression JPEG DCT. |
