### getDeferredLoading() {#getDeferredLoading--}
```
public final boolean getDeferredLoading()
```


Получает или задает значение, указывающее, следует ли загружать дочерние документы явно позже.

--------------------

Значение по умолчанию — `false`, поэтому дочерние документы будут загружаться неявно. Значение `true` указывает, что пользователь должен вызвать `Notebook.loadChildDocument` или для каждого дочернего узла блокнота после загрузки самого блокнота. Если значение `true`, параметр `NotebookLoadOptions.instantLoading` будет игнорироваться. Если блокнот загружается из потока, значение всегда `true`, несмотря на то, что пользователь явно установил его в `false`.

**Returns:**
boolean
### getInstantLoading() {#getInstantLoading--}
```
public boolean getInstantLoading()
```


Получает или задает значение, указывающее, следует ли загружать дочерние документы во время загрузки родительского документа.

--------------------

Значение по умолчанию — `false`, поэтому дочерние документы будут загружаться "лениво", т.е. их загрузка должна откладываться до прямого доступа к конкретному дочернему элементу. Значение `true` указывает, что их загрузка должна происходить немедленно.

**Returns:**
boolean
