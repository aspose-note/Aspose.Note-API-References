## NotebookLoadOptions.DeferredLoading property

Получает или задает значение, указывающее, должны ли дочерние документы загружаться позже явно.

```csharp
public bool DeferredLoading { get; set; }
```

### Примечания

Значение по умолчанию:` false` , поэтому дочерние документы будут загружаться неявно. Значение` true` указывает, что пользователь должен вызывать[`LoadChildDocument`](../../notebook/loadchilddocument)или для дочернего узла каждой записной книжки после загрузки самой записной книжки. Если значение равно` true` ,[`InstantLoading`](../instantloading)опция будет игнорироваться. Если записная книжка загружается из потока, значение всегда равно` true` , несмотря на то, что пользователь явно установил значение` ложь` .

### Примеры

Показывает, как зашифровать блокнот.

```csharp
// Путь к каталогу документов.
string dataDir = RunExamples.GetDataDir_NoteBook();
var notebook = new Notebook(dataDir + "test.onetoc2", new NotebookLoadOptions() { DeferredLoading = true });

notebook.LoadChildDocument(dataDir + "Aspose.one");  
notebook.LoadChildDocument(dataDir + "Locked Pass1.one", new LoadOptions() { DocumentPassword = "pass" });
notebook.LoadChildDocument(dataDir + "Locked Pass2.one", new LoadOptions() { DocumentPassword = "pass2" });
```
