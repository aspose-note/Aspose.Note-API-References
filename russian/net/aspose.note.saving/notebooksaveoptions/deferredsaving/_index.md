## NotebookSaveOptions.DeferredSaving property

Получает или задает значение, указывающее, должны ли дочерние документы сохраняться явно.

```csharp
public bool DeferredSaving { get; set; }
```

### Примечания

Значение по умолчанию:` false` , поэтому дочерние документы будут сохраняться неявно. Значение` true` указывает, что пользователь должен явно сохранять дочерний узел каждой записной книжки. Если записная книжка сохраняется в поток, значение всегда равно` true` , несмотря на то, что пользователь явно установил значение` ложь` .
