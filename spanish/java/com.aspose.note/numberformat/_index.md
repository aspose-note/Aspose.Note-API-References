### LowerLetter {#LowerLetter}
```
public static final byte LowerLetter
```


Especifica que la secuencia debe consistir en una o más ocurrencias de una sola letra del alfabeto latino en minúsculas. Ejemplo: a, b, c, \\u2026, y, z, aa, bb, cc, \\u2026, yy, zz, aaa, bbb, ccc.

### LowerRoman {#LowerRoman}

```
public static final byte LowerRoman
```


Especifica que la secuencia debe consistir en numerales romanos en minúsculas. Ejemplo: i, ii, iii, iv, \\u2026, xviii, xix, xx, xxi.
