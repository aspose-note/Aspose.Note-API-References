## PdfSaveOptions.JpegQuality property

PDF belgesi içindeki JPEG görüntülerinin kalitesini belirleyen bir değer alır veya ayarlar. Değer 0 ile 100 arasında değişebilir; burada 0, en kötü kaliteyi ancak maksimum sıkıştırmayı ve 100, en iyi kaliteyi ancak minimum sıkıştırmayı ifade eder.

```csharp
public int JpegQuality { get; set; }
```

### Notlar

Varsayılan değer 90.

### Örnekler

Belirli ayarlar kullanılarak bir belgenin pdf formatında nasıl kaydedileceğini gösterir.

```csharp
// Belgeler dizininin yolu.
string dataDir = RunExamples.GetDataDir_LoadingAndSaving();

// Belgeyi Aspose.Note'a yükleyin.
Document doc = new Document(dataDir + "Aspose.one");

// PdfSaveOptions nesnesini başlat
PdfSaveOptions opts = new PdfSaveOptions
                          {
                              // Jpeg sıkıştırmasını kullan
                              ImageCompression = Saving.Pdf.PdfImageCompression.Jpeg,

                              // JPEG sıkıştırması için kalite
                              JpegQuality = 90
                          };

dataDir = dataDir + "Document.SaveWithOptions_out.pdf";
doc.Save(dataDir, opts);
```
