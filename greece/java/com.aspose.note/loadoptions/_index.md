
### getLoadHistory() {#getLoadHistory--}

```
public boolean getLoadHistory()
```

Λαμβάνει ή ορίζει μια τιμή που υποδεικνύει εάν ο φορτωτής εγγράφων πρέπει να αγνοεί το ιστορικό. Χρησιμοποιήστε αυτήν την επιλογή για να μειώσετε τη χρήση μνήμης και CPU. Η προεπιλεγμένη τιμή είναι `true`.
