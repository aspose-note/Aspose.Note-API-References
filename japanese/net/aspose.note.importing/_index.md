## クラス

| クラス | 説明 |
| --- | --- |
| [PdfImporter](./pdfimporter/) | PDF 形式のドキュメントからコンテンツをインポートするための API を提供するクラス。[`PdfImportOptions`](../aspose.note.importing/pdfimportoptions/). |
| [PdfImportOptions](./pdfimportoptions/) | PDF ドキュメントからコンテンツをインポートするときにオプションを指定できます。 |
