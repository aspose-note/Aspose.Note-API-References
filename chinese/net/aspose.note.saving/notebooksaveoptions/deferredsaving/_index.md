## NotebookSaveOptions.DeferredSaving property

获取或设置一个值，该值指示是否应明确保存子文档 。

```csharp
public bool DeferredSaving { get; set; }
```

### 评论

默认值为`错误的` , 所以子文档会被隐式保存. 值`真的`表示用户应明确保存每个笔记本的子节点。 如果笔记本正在保存到流，则该值始终`真的`尽管被用户明确设置为`错误的`.
