## NotebookSaveOptions.DeferredSaving property

获取或设置一个值，该值指示是否应显式保存子文档 。

```csharp
public bool DeferredSaving { get; set; }
```

### 评论

默认值为` false` ，因此子文档将被隐式保存。 Value` true` 表示用户应该明确保存每个笔记本的子节点。 如果笔记本正在保存到流，该值始终为` true` 尽管用户明确设置为` 假` 。
