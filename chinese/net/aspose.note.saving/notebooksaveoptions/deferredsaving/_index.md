## NotebookSaveOptions.DeferredSaving property

获取或设置一个值，指示是否应显式保存子文档。

```csharp
public bool DeferredSaving { get; set; }
```

## 备注

默认值为 `false`，因此子文档将隐式保存。值为 `true` 表示用户应显式保存每个笔记本的子节点。如果笔记本保存到流中，无论用户显式设置为 `false`，该值始终为 `true`。
