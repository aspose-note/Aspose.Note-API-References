## NumberFormat enumeration

指定可用于一组自动编号对象的编号格式。完整列表请参阅 [MSDN](https://msdn.microsoft.com/en-us/library/dd923798(v=office.12).aspx)

```csharp
public enum NumberFormat : byte
```

### 值

| 名称 | 值 | 描述 |
| --- | --- | --- |
| DecimalNumbers | `0` | 指定序列应由十进制编号组成。例如：1, 2, 3, …, 8, 9, 10, 11, 12, …, 18, 19, 20, 21。 |
| UpperRoman | `1` | 指定序列应由大写罗马数字组成。例如：I, II, III, IV, …, XVIII, XIX, XX, XXI。 |
| LowerRoman | `2` | 指定序列应由小写罗马数字组成。例如：i, ii, iii, iv, …, xviii, xix, xx, xxi。 |
| UpperLetter | `3` | 指定序列应由一个或多个大写拉丁字母组成。例如：A, B, C, …, Y, Z, AA, BB, CC, …, YY, ZZ, AAA, BBB, CCC。 |
| LowerLetter | `4` | 指定序列应由一个或多个小写拉丁字母组成。例如：a, b, c, …, y, z, aa, bb, cc, …, yy, zz, aaa, bbb, ccc。 |
| TaiwaneseCounting | `33` | 指定序列应由台湾计数系统的顺序数字组成。 |
| TaiwaneseCountingThousand | `35` | 指定序列应由台湾千位计数系统的顺序数字组成。 |
| ChineseCounting | `37` | 指定序列应由中文计数系统的顺序数字组成。 |
| ChineseCountingThousand | `39` | 指定序列应由中文千位计数系统的顺序数字组成。 |
