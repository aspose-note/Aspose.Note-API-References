## Page.PageLayoutSize property

获取或设置页面在编辑器中显示的布局大小。

```csharp
public SizeF PageLayoutSize { get; set; }
```

### 评论

Microsoft OneNote 应用程序使用此值在打开文档时显示基础页面布局。 它不会影响文档的打印和保存。 当 Page.SizeType 属性设置为 PageSizeType.SizeByContent 此属性返回实际大小内容.
