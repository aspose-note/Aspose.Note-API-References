## Page.PageLayoutSize property

获取或设置在编辑器中显示的页面布局大小。

```csharp
public SizeF PageLayoutSize { get; set; }
```

## 备注

此值由 Microsoft OneNote 应用程序用于在打开文档时显示底层页面布局。它不会影响文档的打印和保存。当 Page.SizeType 属性设置为 PageSizeType.SizeByContent 时，此属性返回内容的实际大小。
