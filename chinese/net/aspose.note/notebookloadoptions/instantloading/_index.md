## NotebookLoadOptions.InstantLoading property

获取或设置一个值，该值指示在加载父文档时是否应加载子文档 。

```csharp
public bool InstantLoading { get; set; }
```

### 评论

默认值为`错误的`，因此子文档将“延迟”加载， 即它们的加载应推迟到直接访问特定子文档时。 值`真的`表示应立即完成加载。
