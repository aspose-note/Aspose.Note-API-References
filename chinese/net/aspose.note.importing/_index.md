Importing 命名空间包含提供从 PDF 格式文档导入内容的 API 的类。

## 类

| 类 | 描述 |
| --- | --- |
| [HtmlImporter](./htmlimporter/) | 该类提供用于从 HTML 格式文档导入内容的 API。该 API 允许使用指定的选项从位于文件或流中的 HTML 文档导入。导入选项通过 [`HtmlImportOptions`](../aspose.note.importing/htmlimportoptions/) 传递。 |
| [HtmlImportOptions](./htmlimportoptions/) | 允许在从 HTML 文档导入内容时指定选项。 |
| [PdfImporter](./pdfimporter/) | 该类提供用于从 PDF 格式文档导入内容的 API。该 API 允许使用指定的选项从位于文件或流中的 PDF 文档导入。导入选项通过 [`PdfImportOptions`](../aspose.note.importing/pdfimportoptions/) 传递。 |
| [PdfImportOptions](./pdfimportoptions/) | 允许在从 PDF 文档导入内容时指定选项。 |
