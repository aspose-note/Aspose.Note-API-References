Html 命名空间包含执行保存为 HTML 格式的类。

## 类

| 类 | 描述 |
| --- | --- |
| [CssSavingArgs](./csssavingargs/) | 提供 CssSaving 事件的数据。 |
| [FontSavingArgs](./fontsavingargs/) | 提供 FontSaving 事件的数据。 |
| [ImageSavingArgs](./imagesavingargs/) | 提供 ImageSaving 事件的数据。 |
| [PageSavingArgs](./pagesavingargs/) | 提供 PageSaving 事件的数据。 |
| [ResourceSavingArgs](./resourcesavingargs/) | 提供 ResourceSaving 事件的数据。 |
## 接口

| 接口 | 描述 |
| --- | --- |
| [ICssSavingCallback](./icsssavingcallback/) | 如果您想控制 Aspose.Note 在将文档保存为 HTML 时如何保存 CSS（层叠样式表），请实现此接口。 |
| [IFontSavingCallback](./ifontsavingcallback/) | 如果您想控制 Aspose.Note 在将文档保存为 HTML 时如何保存字体，请实现此接口。 |
| [IImageSavingCallback](./iimagesavingcallback/) | 如果您想控制 Aspose.Note 在将文档保存为 HTML 时如何保存图像，请实现此接口。 |
| [IPageSavingCallback](./ipagesavingcallback/) | 如果您想控制 Aspose.Note 如何保存单独页面，请实现此接口。 |
