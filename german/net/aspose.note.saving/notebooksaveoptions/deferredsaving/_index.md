## NotebookSaveOptions.DeferredSaving property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob untergeordnete Dokumente explizit gespeichert werden sollen.

```csharp
public bool DeferredSaving { get; set; }
```

### Bemerkungen

Standardwert ist`FALSCH` , sodass untergeordnete Dokumente implizit gespeichert werden. Wert`Stimmt`gibt an, dass der Benutzer den untergeordneten Knoten jedes Notebooks explizit speichern sollte. Wenn das Notebook im Stream speichert, ist der Wert immer`Stimmt` obwohl vom Benutzer explizit auf gesetzt wurde`FALSCH` .
