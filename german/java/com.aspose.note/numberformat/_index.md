### DecimalNumbers {#DecimalNumbers}
```

```


Gibt an, dass die Sequenz aus dezimaler Nummerierung bestehen soll. Beispiel: 1, 2, 3, \\u2026, 8, 9, 10, 11, 12, \\u2026, 18, 19, 20, 21.

### LowerLetter {#LowerLetter}
```
public static final byte LowerLetter
```


Gibt an, dass die Sequenz aus einem oder mehreren Vorkommen eines einzelnen Buchstabens des lateinischen Alphabets in Kleinbuchstaben bestehen soll. Beispiel: a, b, c, \\u2026, y, z, aa, bb, cc, \\u2026, yy, zz, aaa, bbb, ccc.

### LowerRoman {#LowerRoman}

public static final byte LowerRoman

Gibt an, dass die Sequenz aus römischen Ziffern in Kleinbuchstaben bestehen soll. Beispiel: i, ii, iii, iv, \\u2026, xviii, xix, xx, xxi.
