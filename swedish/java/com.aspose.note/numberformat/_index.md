### LowerLetter {#LowerLetter}
```
public static final byte LowerLetter
```


Anger att sekvensen ska bestå av en eller flera förekomster av en enskild bokstav i det latinska alfabetet i gemener. Exempel: a, b, c, \\u2026, y, z, aa, bb, cc, \\u2026, yy, zz, aaa, bbb, ccc.

### LowerRoman {#LowerRoman}
```
public static final byte LowerRoman
```


Anger att sekvensen ska bestå av romerska siffror i gemener. Exempel: i, ii, iii, iv, \\u2026, xviii, xix, xx, xxi.
