## NotebookLoadOptions.InstantLoading property

Ottiene o imposta un valore che indica se i documenti figli devono essere caricati durante il caricamento del documento padre.

```csharp
public bool InstantLoading { get; set; }
```

### Osservazioni

Il valore predefinito è`falso` , quindi i documenti figlio verranno caricati "pigramente", cioè il loro caricamento dovrebbe essere posticipato fino all'accesso diretto a un figlio specifico. Valore`VERO` indica che il caricamento deve essere eseguito immediatamente.
